# Frozen Protocol Summary

The benchmark evaluates target prioritization under a held-out-herb setting. The split, target namespace, metric contract, final model set, final seeds, and reporting rules were frozen before the one-time sealed-test evaluation.

The public release contains only aggregate results and public source data. It does not contain held-out labels, target-attributable held-out records, prediction-level outputs, score matrices, or checkpoints.

Primary reporting is benchmark-first. The repository reports the frozen aggregate ranking and descriptive differences without using sealed-test outcomes for additional model selection.
