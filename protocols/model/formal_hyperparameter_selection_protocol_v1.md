# Formal Hyperparameter Selection Protocol v1

Each of five models receives exactly CFG01-CFG04. No third layer, model-specific learning rate, test-driven candidate, or runtime grid extension is allowed. RGCN bases are min(4, relation_count); TCMRGAT uses four heads because the legacy architecture supports multi-head attention. A resource-infeasible configuration is reported without substitution; any common downscale requires a new protocol version before D2.
