# Source terms and attribution guide

This file records the source terms applicable to materials used to construct the frozen benchmark. It is an operational attribution guide, not legal advice. Consult the provider's current licence page before redistribution or a new use.

| Source | Relevant material | Declared source term | Official information |
| --- | --- | --- | --- |
| Dryad | Herb-related source records | CC0 for Dryad datasets | https://v3.datadryad.org/help/guides/reuse |
| ChEMBL | Compound-target context | CC BY-SA 3.0 | https://chembl.gitbook.io/chembl-interface-documentation/frequently-asked-questions/general-questions |
| HGNC | Gene nomenclature and identifiers | CC0 | https://www.genenames.org/download/statistics-and-files/ |
| UniProt | Protein identifiers and related records | CC BY 4.0 | https://www.uniprot.org/help/license |
| STRING | Target-target context | CC BY 4.0 | https://string-db.org/cgi/access?footer_active_subpage=licensing |

The frozen graph is a mixed-source-derived artifact. It is supplied to permit reproduction of this benchmark, not to replace the original providers. Users who redistribute or adapt material from this package must preserve attribution and satisfy any applicable ShareAlike requirement, especially for ChEMBL-derived database content.

For the source location of the Dryad herb-related records, see https://datadryad.org/dataset/doi%3A10.5061/dryad.wh70rxwx9.
