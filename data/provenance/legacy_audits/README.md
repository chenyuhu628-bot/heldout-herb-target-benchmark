# Status of legacy review-safe audits

The files in this directory were created for the earlier review-safe release, in which the fixed test labels, full score matrices, and selected checkpoint files were not publicly available.

They are preserved as historical protocol and review evidence only. Their references to restricted, controlled, sealed, or review-safe access do not describe the availability of this v1.2 release. For the current public scope, use README.md, data_availability/DATA_AVAILABILITY_v1.2.md, protocols/open_release_evaluation_protocol_v1.md, and audits/open_release_audit_v1.md.
