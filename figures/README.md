# Figures and source data

The figure files are publication-supporting outputs for the associated manuscript. Their source data are in figures/source_data.

Supplementary Figure S2 was regenerated for v1.2. It presents the public independent-evaluation route: fixed split information, 8,632 public recorded-positive test labels for 67 herbs, the 2,309-target universe, frozen graph inputs, 25 archived checkpoints, and 25 complete score matrices. Its source data are in figures/source_data/supplementary_figure_S2_source_data.csv.

Supplementary Figures S3 through S5 and their source-data files retain historical result displays. Their source-data notes now distinguish historical sealed-evaluation provenance from current v1.2 public availability. The current access-scope statements are in README.md, data_availability/DATA_AVAILABILITY_v1.2.md, and audits/open_release_audit_v1.md.

Figure and table narrative text should be updated to cite the v1.2 version-specific archive DOI once it is assigned.
